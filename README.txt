シンプルな３D空間シューティングです。
単純なポリゴンデータを読み込んで自機を表示します。
ソケットプログラミングの確認の為に簡易サーバープログラムも作成しました。

使用言語：C,C++,(グラフィックライブラリとして)OpenGL

【内容】
    dat/
	自機に使用されるポリゴンデータファイル（dat1.pol）
	通信用IPアドレスファイル（ip.txt）
    src／
	class／ネットワーク通信用クラスソースファイル
	Gameprog／ゲームプログラミング部分ソースファイル
	Winsprog／ウィンドウズ部分ソースファイル
	GL_Utility／OpenGL関連ヘッダー、ライブラリファイル
    
    EasyServer／通信対戦用サーバープログラム
	src／サーバ部分ソース・ヘッダーファイル
